# alt-ime-mapper

AutoHotkey v2 で動作する、左右 Alt キーの空打ちによる IME 切り替えツールです。

## 特徴

- **AHK v2 専用**: 最新の AutoHotkey v2.0+ で動作します。
- **モダンアプリ対応**: `ControlGetFocus` を用いたハンドル取得により、新しいメモ帳 (Notepad) 等のUWPアプリでも動作します。
- **ショートカットを邪魔しない**: `~` プレフィックスと `vkE8` マスクキーの採用により、`Alt + Tab` や `Alt + F4` などの既存ショートカットを阻害しません。
- **メニューフォーカス防止**: Alt 単体押し時にメニューバーがアクティブになる挙動を抑制します。

## 使い方

- **左 Alt キー空打ち**: IME を **OFF** にします。
- **右 Alt キー空打ち**: IME を **ON** にします。
- 他のキーと組み合わせて押した場合は、通常の Alt キーとして動作します。

## セットアップ

1. [AutoHotkey v2](https://www.autohotkey.com/) をインストールします。
2. `alt-ime-mapper.ahk` を任意の場所に保存します。
3. `alt-ime-mapper.ahk` を実行します。

スタートアップにショートカットを作成することで、スタートアップ時に自動で起動するようになります。
